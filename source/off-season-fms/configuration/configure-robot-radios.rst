Configure Robot Radios
======================

Get those bots ready for play!

Instructions
------------

Teams should use the FRC\ |reg| bridge configuration utility to program their robot radios for Off-Season use. Please see `this article <https://docs.wpilib.org/en/stable/docs/software/driverstation/programming-radios-for-fms-offseason.html>`_ for instructions.
